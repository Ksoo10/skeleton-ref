Design Tokens — :root

}

Layout — Wrappers & Containers
/* Center content horizontally with side margins */
.standard-block {
  margin: 0.75rem 1rem 1rem 1rem;
}

/* Stacks vertically on mobile (<768px) */
.wrapper {
  margin: 1rem 0;
}

/* Centered text/content block */
.text-wrapper {
  margin: 0 auto;
  padding: 0.5rem;
}

/* Generic auto-centered container */
.container {
  margin-left: auto;
  margin-right: auto;
}

/* Section default */
section {
  margin: 0 auto;
  padding: 2rem 1rem;
}

Combine .container with a width utility like .block-650 to constrain content width.

Accordion — .details-wrapper
Live Preview

FAQ Section

What is this accordion for?

It opens and reveals content when clicked. Uses native HTML details/summary — no JS needed.

How does the open state look?

The background switches to an oklch orange tint using details[open].

.details-wrapper {
  background: #ffffff;
  border-radius: 16px;
  padding: 1rem;
}
.detail-title-section {
  background: var(--clr-secondary);
  border-radius: 8px;
  padding: 1rem;
}
details[open] {
  background: oklch(95% 0.1812 38.35);
}
<!-- HTML Skeleton -->
<div class="details-wrapper">
  <div class="detail-title-section">
    <h3>Section Title</h3>
  </div>
  <details>
    <summary>Question / Label</summary>
    <p>Answer content here.</p>
  </details>
</div>

  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />

        &family=Unbounded:wght@200..900&display=swap" rel="stylesheet" />
  <link rel="stylesheet" href="style.css" />
</head>
<body>

  <header>
    <nav class="menu container">
      <ul class="flex items-center gap-4">
